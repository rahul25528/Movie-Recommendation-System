## 🧠 Techniques Used

### Method 1 — Weighted Rating (IMDB Formula)
Provides a fair global ranking by balancing rating quality with vote quantity.

```
Score = (v / (v + m)) × R + (m / (v + m)) × C
```

| Symbol | Meaning |
|--------|---------|
| v | Vote count for the movie |
| R | Average rating of the movie |
| m | Minimum votes threshold (80th percentile = 51) |
| C | Mean rating across all movies (5.64) |

### Method 2 — Content-Based Filtering (TF-IDF + Cosine Similarity)
Finds movies similar to a given title by analyzing plot description and genres.

- TF-IDF matrix: **44,506 × 15,000** features
- Genres weighted **2×** for stronger genre signal  
- Similarity: cosine distance

### Method 3 — Hybrid Recommender
Combines both approaches for best results:

```
hybrid_score = 0.6 × content_similarity + 0.4 × weighted_score
```

## 💡 Usage Examples

```python

    get_top_movies,
    get_content_recommendations,
    get_hybrid_recommendations
)

get_top_movies(top_n=10)

get_top_movies(genre='Action', year_from=2000, year_to=2020)

get_hybrid_recommendations('Toy Story', top_n=10)
```

## 🌐 Web App API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/` | GET | Web UI |
| `/api/top` | GET | Top movies with filters |
| `/api/recommend` | GET | Movie recommendations |
| `/api/search` | GET | Autocomplete search |
| `/api/stats` | GET | Dataset statistics |

**Query parameters for `/api/top`:**
- `genre` — filter by genre (e.g. Action, Drama)
- `year_from` / `year_to` — year range
- `top_n` — number of results (max 20)

**Query parameters for `/api/recommend`:**
- `title` — movie title (required)
- `method` — `hybrid` or `content`
- `top_n` — number of results (max 20)

| `popularity` | Popularity score |
